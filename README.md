# ConchShell
Custom linux shell implementation written in C
Inspired by classic shells, but written with the voice of the ocean

## Features
- basic shell loop
- custom prompt: `conch>`
- runs external commands
- built in commands: `exit`, `cd`, `ask`, `help`
